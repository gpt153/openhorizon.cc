# Document Export - Application-Ready Documents

## What is Document Export?

Export transforms your OpenHorizon project into polished, submission-ready documents in multiple formats. Your work becomes professional application materials aligned with Erasmus+ requirements.

## Export Formats

### PDF (Portable Document Format)

**Best for:**
- Final submissions
- Sharing with partners (read-only)
- Printing
- Archiving

**Includes:**
- Professional formatting
- Page numbers and table of contents
- Embedded images and charts
- Gantt chart visualizations
- Budget tables
- All project sections

**Use when:**
- Submitting to National Agency
- Presenting to stakeholders
- Creating permanent records
- Partners don't need to edit

### Word (DOCX)

**Best for:**
- Collaborative editing
- Partner input
- Form completion
- Final adjustments

**Includes:**
- Editable text and tables
- Formatted budget spreadsheets
- Programme schedules
- Application forms
- Comments enabled

**Use when:**
- Partners need to add content
- Forms require manual completion
- Last-minute changes likely
- National Agency requests Word format

### Excel (XLSX)

**Best for:**
- Budget analysis
- Financial reporting
- Data manipulation
- Detailed planning

**Includes:**
- Budget breakdown (all categories)
- Participant lists
- Timeline data
- Cost calculations
- Vendor comparisons

**Use when:**
- Finance team needs budget details
- Comparing scenarios
- Reporting to funders
- Detailed cost analysis

## What's Included in Exports

### All Exports Contain:

**1. Project Overview**
- Formal and informal titles
- Full descriptions
- Target group details
- Learning objectives
- Expected outcomes

**2. Organization Information**
- Organization name and details
- Contact information
- Legal status
- Previous experience

**3. Partner Information** (if applicable)
- Partner organizations
- Roles and responsibilities
- Contact details

**4. Project Timeline**
- Gantt chart (visual in PDF, data in Excel)
- Phase breakdown
- Key milestones
- Start and end dates

**5. Programme**
- Day-by-day schedules
- Activity details (time, location, description)
- Facilitators and materials
- Full activity breakdown

**6. Participants**
- Total numbers
- Age ranges
- Countries represented
- Special needs accommodation
- Group leader information
- Fewer opportunities groups

**7. Budget**
- Category breakdown (travel, organisational, individual support)
- Unit cost calculations
- Total budget
- Co-funding requirements
- Inclusion support
- Exceptional costs (if any)

**8. Erasmus+ Application Forms** (PDF/Word)
- KA1 (Learning Mobility) forms
- KA2 (Cooperation Partnerships) forms
- Pre-filled with project data
- Ready for signatures

**9. Appendices**
- Vendor quotes (if uploaded)
- Letters of intent from partners
- Supporting documents

## How to Export

### Step 1: Prepare Your Project

Before exporting, ensure:

- [ ] All required sections completed
- [ ] Descriptions reviewed for clarity
- [ ] Budget verified and realistic
- [ ] Programme finalized
- [ ] Partner information accurate
- [ ] Contact details current
- [ ] Dates confirmed

**Quick check:**
- Click **"Review"** button
- System highlights missing information
- Complete any gaps

### Step 2: Choose Export Format

1. Click **"Export"** in top menu
2. Select format:
   - **"Export to PDF"** - Read-only, professional
   - **"Export to Word"** - Editable, collaborative
   - **"Export to Excel"** - Budget and data analysis
   - **"Export All"** - Get all three formats

3. Choose what to include:
   - ‚òë Project overview and descriptions
   - ‚òë Timeline and Gantt chart
   - ‚òë Detailed programme
   - ‚òë Budget breakdown
   - ‚òë Participant information
   - ‚òë Partner details
   - ‚òë Application forms (KA1/KA2)
   - ‚òë Appendices (quotes, letters)

### Step 3: Customize (Optional)

**Document settings:**
- **Language** - Choose output language (if available)
- **Template** - Standard, detailed, or minimal
- **Branding** - Include organization logo
- **Headers/Footers** - Add custom text

**Privacy options:**
- Exclude participant personal data (names, contact info)
- Anonymize sensitive information
- Remove internal notes and comments

### Step 4: Download

1. Click **"Generate Export"**
2. Wait 10-30 seconds (depending on project size)
3. Download appears
4. Save to your computer

**File naming:**
```
[Project-Title]_[Format]_[Date].pdf
Example: Digital-Skills-Youth_PDF_2026-01-17.pdf
```

## Erasmus+ Application Forms

### KA1 (Learning Mobility)

Auto-generated form includes:

**Section A: Organization Information**
- Legal name and ID
- Address and contacts
- Legal representative
- Bank details (add manually)

**Section B: Project Description**
- Context and objectives
- Number and profile of participants
- Preparation activities
- Activity description (pre-filled from programme)
- Follow-up activities
- Budget breakdown

**Section C: Impact**
- Expected learning outcomes
- Dissemination plans
- Sustainability

**Your role:**
- Review pre-filled sections
- Add signatures
- Upload to EU portal
- Attach supporting documents

### KA2 (Cooperation Partnerships)

Auto-generated form includes:

**Section 1: Context**
- Relevance to programme priorities
- Needs analysis
- Target groups

**Section 2: Project Design**
- Objectives and expected outcomes
- Methodology (pre-filled from project)
- Activities and timeline
- Partnership composition

**Section 3: Budget**
- Full budget breakdown
- Co-financing plan
- Cost justification

**Section 4: Impact**
- Dissemination and exploitation
- Sustainability

**Your role:**
- Review and refine content
- Ensure partner alignment
- Get partner signatures
- Submit via EU portal

## After Export

### Review Checklist

Before submission, verify:

- [ ] All sections complete and accurate
- [ ] No placeholder text (e.g., "TBD", "[To be added]")
- [ ] Budget totals match across sections
- [ ] Dates are consistent throughout
- [ ] Partner details are correct
- [ ] Contact information current
- [ ] Spelling and grammar checked
- [ ] Page numbers correct
- [ ] Formatting professional

### Partner Review

Share with partners for:
- Content verification
- Accuracy check
- Role confirmation
- Final approval

**Allow time:**
- Partners need 1-2 weeks for review
- Revisions may be needed
- Approval signatures required
- Don't wait until deadline!

### Final Submission

**Before submitting:**
1. Save final version with date
2. Keep backup copies
3. Export to PDF for submission
4. Gather required signatures
5. Prepare supporting documents
6. Check National Agency requirements

**Submission checklist:**
- [ ] Main application form (PDF)
- [ ] Partner letters of intent
- [ ] Organization legal documents
- [ ] Budget justification (if required)
- [ ] Participant consent forms (if required)
- [ ] Additional documents per National Agency

## Re-Exporting

You can export multiple times:

**When to re-export:**
- After making changes
- When adding new information
- For different audiences (detailed vs. minimal)
- To get updated application forms
- After partner feedback

**Version control:**
- Exports are timestamped
- Keep previous versions for reference
- Name files clearly: `Project_v1.pdf`, `Project_v2.pdf`, `Project_FINAL.pdf`

## Troubleshooting Exports

### Export Fails or Hangs

**Try:**
1. Refresh the page
2. Check all required fields are complete
3. Try different format (PDF vs. Word)
4. Clear browser cache
5. Use different browser

### Downloaded File is Blank

**Possible causes:**
- Export still processing (wait and try again)
- Browser popup blocker (disable for OpenHorizon)
- File corrupted (re-export)

### Budget Numbers Don't Match

**Check:**
- Budget calculator is up to date
- Participant numbers consistent across sections
- Timeline dates match budget period
- Inclusion costs are included/excluded as expected

### Missing Content

**Verify:**
- All sections of project are saved
- Programme is finalized
- Budget calculation is complete
- Export template includes desired sections

## Common Questions

**Q: Can I edit exports after downloading?**
A: Word and Excel files are fully editable. PDF can be edited with PDF tools but better to edit in OpenHorizon and re-export.

**Q: Do I need to export before submitting?**
A: Yes. Erasmus+ applications are submitted as documents, not direct from OpenHorizon.

**Q: Can I customize the export format?**
A: Limited customization (templates, branding). For major changes, export to Word and edit.

**Q: Are exports saved in OpenHorizon?**
A: No. Downloads are not stored. You can re-export anytime.

**Q: How do I export in a different language?**
A: Project content is in the language you wrote it. Translation features are planned.

**Q: Can partners export directly?**
A: If they have project access, yes. Otherwise, share exported files.

**Q: Is there a file size limit?**
A: Typically 10-20 MB per export. Very large programmes may need detailed export split.
